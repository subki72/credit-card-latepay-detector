# credit-card-latepay-detector
FinanKu Credit Default Prediction  Proyek ini membangun model pembelajaran mesin untuk memprediksi risiko keterlambatan pembayaran kartu kredit menggunakan data nasabah dari FinanKu. Berfokus pada analisis perilaku keuangan, fitur engineered dari transaksi dan saldo kuartal, serta evaluasi model berbasis recall. Hasilnya diharapkan mampu membantu tim bisnis untuk mengurangi risiko gagal bayar dan merancang strategi penanganan nasabah berisiko tinggi.  Fitur:  Eksplorasi & preprocessing data nasabah  Feature engineering saldo, produk, dan umur kartu  Modelling dengan Logistic Regression, Random Forest, XGBoost  Evaluasi model: recall, accuracy, dan confusion matrix  Saran perbaikan imbalance data dan optimasi model
